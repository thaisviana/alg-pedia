The AlgPedia is a free encyclopedia that is collaboratively constructed in order to provide an RDF-based algorithms that can be accessed via the website or through Open Linked Data applications that process structured data.
This site is based on the concept Wiki Wiki, one of the most famous Web 2.0 applications, which encourages the masses to contribute, enabling the creation of new pages and edit existing content. This model of knowledge construction is a manifestation of collective intelligence, an intelligence distributed everywhere, constantly enhanced, coordinated in real time, which results in an effective mobilization of skills.
However, the structure of the current web allows AlgPedia not be accessed by programs and knowledge processed by an automated intelligent agents, it is necessary to add a set of metadata to documents encyclopedia.
To add meaning to content provided by our application we modeled an ontology that describes our algorithms base providing additional vocabulary along with a formal semantics.
